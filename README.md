# 🏢 Job Portal Backend API

A **Job Portal Backend API** built with Spring Boot that enables seamless interaction between **Job Seekers** and **Recruiters**.  
This project demonstrates **JWT-based authentication**, **REST API best practices**, and **search with pagination & sorting**.

---

## ✨ Features

- 👤 **Users**
  - **Job Seekers** can browse job listings and apply to jobs.
  - **Recruiters** can post new jobs and manage existing job postings.
- 🔑 **JWT Authentication** for secure login & access control.
- 🔍 **Search, Pagination & Sorting** for efficient job listing retrieval.
- 📑 **Swagger API Docs** for easy testing & exploration.
- 🛠️ **Postman Collection** included for API testing.

---

## 🔧 Tech Stack

- **Spring Boot** – RESTful API development  
- **Spring Security + JWT** – Authentication & Authorization  
- **Spring Data JPA** – Database interactions  
- **Swagger** – API documentation  
- **Postman** – API testing & collection sharing  
- **GitHub** – Version control & project hosting
